Componente: Tarjetas Dinámicas

Colección de tarjetas reutilizables con animaciones de entrada, volteo al clic, microinteracciones por botón (favorito, carrito, expandir) y reverso con descripción y botón Comprar. Personaliza colores, tamaños y comportamiento desde CSS y JS.

Implementación rápida

Este componente está pensado para integrarse en cualquier página web con tres pasos:

  1. Incluir estilos y scripts: copia css/componente.css y css/index.css a tu proyecto y añade las etiquetas <link> en el <head>. Incluye js/componente.js antes de tu script que crea las tarjetas (por ejemplo js/index.js).
  2. Crear un contenedor: añade un elemento en el HTML donde quieras insertar las tarjetas, por ejemplo: 
    <div id="catalogo"></div>
  3. Generar tarjetas desde JavaScript: llama a crearTarjeta({...}) pasando las opciones necesarias. Ejemplo mínimo: 
    crearTarjeta({
  contenedorId: "catalogo",
  imagen: "img/producto.webp",
  titulo: "Nombre del producto",
  precio: "$199.00",
  descripcion: "Descripción breve",
  animacion: "animacion-arriba",
  variante: "cuadrada",
  estado: "nuevo",
  estiloInline: "--color-fondo:#fff7f0;",
  tilt: true
});

Eventos disponibles: el componente emite eventos personalizados para integrar lógica externa:

  • favoritoToggle — se dispara cuando el usuario marca/desmarca favorito.
  • agregarCarrito — se dispara al pulsar el botón carrito.
  • expandirDetalles — se dispara al pulsar expandir (ideal para redirecciones o abrir modal).
  • comprarProducto — se dispara al pulsar Comprar en el reverso.

Ejemplo de escucha global: 

document.addEventListener('agregarCarrito', e => {
  console.log('Agregar al carrito:', e.detail.titulo);
});

Personalización: modifica las variables en :root dentro de componente.css para cambiar colores, tamaños y estados. Usa la opción estiloInline para aplicar variables CSS por tarjeta.
